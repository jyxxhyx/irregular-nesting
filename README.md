# 面料裁切利用率优化比赛

启发式方法求解面料裁切问题。

## 使用

运行`/code/main.py`即可，程序会遍历`/data`文件夹下各文件夹（如DatasetA）的算例进行求解，结果存在`/submit`文件夹中同名文件夹下。

### 依赖项（由`pipreqs`生成）

  matplotlib==3.1.1
